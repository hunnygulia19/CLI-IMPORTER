\# Data Import CLI



A Command-Line Interface (CLI) tool to import CSV or JSON files and insert them into \*\*MySQL\*\* or \*\*MongoDB\*\* databases.



\## Features



\- Import CSV or JSON files.

\- Insert data into MySQL tables or MongoDB collections.

\- Preview data after loading.

\- Simple and interactive CLI interface.



\## Installation



1\. Clone the repository:

```bash

git clone <your-repo-url>

cd <your-repo-folder>



2\\. pip install -r requirements.txt



\\## Usage



1\\. Run the CLI:

```bash

python main.py





2\\. Select the file type:



1\\. CSV

2\\. JSON



3\\. Enter the path to your file.



4\\. Choose the database to insert data:



1\\. MySQL

2\\. MongoDB



5\\. Enter the table name (for MySQL) or collection name (for MongoDB).



6\\. Data will be inserted into the selected database, and a success message                     will be shown.
